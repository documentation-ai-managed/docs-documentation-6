<Update label="2024-11-01" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Introduce AI-assisted content generation for faster doc creation.
- Add real-time collaboration with live editing and comments.

## Improvements

- Optimize rendering speed by `>40%` for large MDX files.

## Bug Fixes

- Resolve parsing errors with nested `{` and `}` characters.

## Breaking Changes

- Update default theme color to `#3B82F6`. Migrate custom CSS if needed.

```javascript
// Before (v1.x)
const theme = { primary: '#1E40AF' };

// After (v2.0+)
const theme = { primary: '#3B82F6' };
```

</Update>
